# Security rules

## Current phase

Shadow monitoring only.

## Hard prohibitions

- No private key in Railway.
- No private key in GitHub.
- No autonomous execution.
- No unlimited approvals.
- No production contract deployment.
- No flash-loan logic.
- No cross-DEX arbitrage logic.

## Future execution prerequisites

Execution can be discussed only after:

1. scanner works for at least 7 days;
2. VPS runs for at least 48 hours without crash;
3. real Morpho scanner is validated;
4. simulator proves candidate profitability;
5. hot wallet limits are implemented;
6. kill switch is implemented;
7. contracts are reviewed.
